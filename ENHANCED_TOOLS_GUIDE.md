# Enhanced Tools Guide - Chemical Reaction Drawer

## Overview

The Chemical Reaction Drawer now includes a comprehensive set of drawing tools for creating professional chemical diagrams, reaction schemes, and molecular structures.

---

## 🎨 Tool Categories

### 1. Basic Tools
- **Atom Tool (A)** - Place atoms with element selection
- **Selection Tool (S)** - Select and move objects

### 2. Text Tools
- **Text Box (T)** - Add text annotations and labels

### 3. Bond Drawing Tools

#### Standard Bonds
- **Single Bond (1)** - Draw single covalent bonds (─)
- **Double Bond (2)** - Draw double covalent bonds (═)
- **Triple Bond (3)** - Draw triple covalent bonds (≡)

#### Stereochemical Bonds
- **Front Bond / Wedge (W)** - Solid wedge for bonds coming out of plane (▶)
- **Back Bond / Dash (D)** - Dashed wedge for bonds going behind plane (⋯)
- **Unknown Bond / Wavy (U)** - Wavy line for unknown stereochemistry (〰)

### 4. Reaction Arrows
- **Reaction Arrow (R)** - Standard reaction arrow (→)
- **Equilibrium Arrow** - Double arrow for equilibrium reactions (⇌)
- **Resonance Arrow** - Double-headed arrow for resonance structures (↔)

### 5. Shape Tools
- **Triangle** - Draw triangular shapes (△)
- **Square** - Draw square shapes (□)
- **Pentagon** - Draw pentagonal shapes (⬠)
- **Hexagon** - Draw hexagonal shapes (⬡)
- **Heptagon** - Draw heptagonal shapes (⬢)
- **Octagon** - Draw octagonal shapes (⬣)

### 6. Ring Templates
- **Benzene Ring (B)** - Hexagon with alternating double bonds
- **Benzyl Ring** - Hexagon with 3 double bonds

### 7. Eraser Tools
- **Circular Eraser (E)** - Erase with circular brush (⭕)
- **Square Eraser** - Erase with square brush (⬜)
- **Random Eraser** - Erase with random pattern (🎲)

### 8. Layout Tools
- **Table** - Insert tables for data organization (⊞)

## 🎯 Quick Start

### Using Bond Tools

1. **Single Bond:**
   - Press `1` or click "Single Bond"
   - Click and drag between two atoms
   - Release to create bond

2. **Double Bond:**
   - Press `2` or click "Double Bond"
   - Click and drag between atoms
   - Creates parallel lines for double bond

3. **Stereochemical Bonds:**
   - Press `W` for wedge (front)
   - Press `D` for dash (back)
   - Press `U` for wavy (unknown)
   - Click and drag to draw

### Using Shape Tools

1. **Select Shape:**
   - Click desired shape (Triangle, Square, etc.)
   
2. **Draw Shape:**
   - Click on canvas where you want center
   - Shape appears at default size
   
3. **Adjust Size:**
   - Use "Shape Size" slider in settings
   - Range: 30-300 pixels

### Using Ring Templates

1. **Benzene Ring:**
   - Press `B` or click "Benzene Ring"
   - Click on canvas
   - Creates hexagon with alternating double bonds
   
2. **Benzyl Ring:**
   - Click "Benzyl Ring"
   - Click on canvas
   - Creates hexagon with 3 double bonds

### Using Erasers

1. **Circular Eraser:**
   - Press `E` or click "Circular Eraser"
   - Click and drag to erase
   - Adjust size with "Eraser Size" slider
   
2. **Square Eraser:**
   - Click "Square Eraser"
   - Click and drag to erase in square pattern
   
3. **Random Eraser:**
   - Click "Random Eraser"
   - Click to erase with random pattern

### Using Reaction Arrows

1. **Reaction Arrow:**
   - Press `R` or click "Reaction Arrow"
   - Click and drag from reactant to product
   - Creates arrow with arrowhead
   
2. **Equilibrium Arrow:**
   - Click "Equilibrium Arrow"
   - Click and drag
   - Creates double arrow (forward and reverse)
   
3. **Resonance Arrow:**
   - Click "Resonance Arrow"
   - Click and drag
   - Creates double-headed arrow

### Using Text Tool

1. **Add Text:**
   - Press `T` or click "Text Box"
   - Click where you want text
   - Type your text
   - Press Enter to confirm
   
2. **Adjust Font Size:**
   - Use "Text Size" slider
   - Range: 8-72 points

## 💡 Tips & Best Practices

### Drawing Molecules

1. **Start with Benzene Ring:**
   - Use Benzene Ring tool (B) for aromatic compounds
   - Faster than drawing individual atoms and bonds

2. **Use Stereochemical Bonds:**
   - Wedge bonds (W) for groups coming toward you
   - Dash bonds (D) for groups going away
   - Wavy bonds (U) when stereochemistry is unknown

3. **Adjust Bond Lengths:**
   - Use Bond Length slider for consistent spacing
   - Longer bonds for clarity in complex structures

### Drawing Reactions

1. **Reaction Arrows:**
   - Use Reaction Arrow (R) for standard reactions
   - Use Equilibrium Arrow for reversible reactions
   - Add text above/below arrows for conditions

2. **Multi-Step Reactions:**
   - Draw each step separately
   - Use multiple reaction arrows
   - Label each step with text tool

### Using Shapes

1. **Highlighting:**
   - Use shapes to highlight important parts
   - Circles for functional groups
   - Squares for reaction centers

2. **Mechanism Arrows:**
   - Combine arrows with shapes
   - Show electron movement
   - Indicate reaction pathways

### Erasing

1. **Precision Erasing:**
   - Use Circular Eraser (E) for precise control
   - Adjust size for fine details

2. **Quick Cleanup:**
   - Use Square Eraser for larger areas
   - Random Eraser for artistic effects

## 🔧 Advanced Features

### Custom Tool Combinations

1. **Complex Molecules:**
   - Start with ring template
   - Add bonds with bond tools
   - Place atoms at bond ends
   - Add stereochemistry as needed

2. **Reaction Schemes:**
   - Draw reactants
   - Add reaction arrow
   - Draw products
   - Add text for conditions
   - Use shapes for emphasis

3. **Mechanism Diagrams:**
   - Draw starting structure
   - Add curved arrows (use arrow tools)
   - Show intermediate structures
   - Indicate electron movement

### Workflow Tips

1. **Template First:**
   - Use ring templates for common structures
   - Modify with bond and atom tools
   - Faster than building from scratch

2. **Layer Your Drawing:**
   - Draw main structure first
   - Add stereochemistry second
   - Add annotations last
   - Use text tool for labels

3. **Use Grid:**
   - Enable "Show Grid" for alignment
   - Enable "Snap to Grid" for precision
   - Adjust grid size in settings

## 🐛 Troubleshooting

### Tool Not Working?

1. **Check Tool Selection:**
   - Ensure correct tool is highlighted
   - Look for blue highlight on button

2. **Check Canvas Focus:**
   - Click on canvas area
   - Try tool again

3. **Check Settings:**
   - Verify size settings are reasonable
   - Reset to defaults if needed

### Bonds Not Connecting?

1. **Atom Proximity:**
   - Click near existing atoms
   - Atoms snap within 20 pixels

2. **Bond Tool Selected:**
   - Ensure bond tool is active
   - Check bond type is correct

### Shapes Not Appearing?

1. **Size Setting:**
   - Check Shape Size slider
   - Increase if too small

2. **Canvas Bounds:**
   - Ensure clicking within canvas
   - Check zoom level

## 📚 Examples

### Example 1: Drawing Aspirin

```
1. Press B - Draw benzene ring
2. Press 1 - Add single bond from ring
3. Press A - Add C atom
4. Press 2 - Add double bond
5. Press A - Add O atom
6. Press T - Add "COOH" text
7. Press 1 - Add bond to other position
8. Press A - Add O atom
9. Press T - Add "OCOCH3" text
```

### Example 2: Reaction Scheme

```
1. Draw reactant molecule
2. Press R - Draw reaction arrow
3. Press T - Add conditions above arrow
4. Draw product molecule
5. Use shapes to highlight changes
```

### Example 3: Stereochemistry

```
1. Draw carbon center
2. Press W - Add wedge bond (front group)
3. Press D - Add dash bond (back group)
4. Press 1 - Add normal bonds (in-plane groups)
5. Press A - Add atoms to bond ends
```
